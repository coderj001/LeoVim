# LeoVim
> neovim, lua, neovim-plugn-lua

### How to run

1. Run build command `docker build -t leovim .`
2. Run the docker engine  `docker run -it --rm -v $(pwd):/root/.config/nvim/ leovim`
3. Inside shell run `nvim` and install plugins by `:PackerInstall`
